# What is Supervised Learning?

## Learning with Answers

Supervised learning is like studying with an answer key. You learn from examples where you know the correct answers, then apply that knowledge to new examples.

### The Concept

Imagine you're learning to identify animals:

**Training Data (with answers):**
- Picture of a dog → Label: "Dog"
- Picture of a cat → Label: "Cat"
- Picture of a bird → Label: "Bird"

**Goal:** Learn the pattern so you can identify new animals without labels.

### Features and Labels

- **Features**: The input data (measurements, pixels, text, etc.)
- **Labels**: The correct answers you want to predict

### Types of Supervised Learning

1. **Regression**: Predict continuous values
   - House prices, temperature, stock prices

2. **Classification**: Predict categories
   - Spam/not spam, dog/cat/bird, pass/fail

### The Learning Process

1. **Collect labeled data**
2. **Choose a model** (linear regression, decision trees, neural networks)
3. **Train the model** on known examples
4. **Test on new data** to see how well it learned
5. **Use the model** to make predictions

### Real-World Examples

- Email spam detection
- Handwriting recognition
- Medical diagnosis
- Stock price prediction
- Language translation

### Remember
- Requires labeled training data
- Goal is to generalize from examples
- Performance measured on unseen data
- Foundation of many AI applications
