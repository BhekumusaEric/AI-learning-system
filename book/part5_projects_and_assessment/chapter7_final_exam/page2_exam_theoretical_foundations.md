# Section 1: Theoretical Foundations

## Machine Learning & Neural Network Theory

### Instructions
Answer all questions in this section. Each question is worth 10 points. Provide clear, concise explanations with mathematical notation where appropriate. Show your work for calculations.

---

## Question 1: Machine Learning Fundamentals (20 points)

**a) Explain the difference between supervised, unsupervised, and reinforcement learning. Give a real-world example of each.**

**b) What is the bias-variance tradeoff? How does it affect model selection?**

**c) Describe the k-fold cross-validation technique. What are its advantages over simple train/test splits?**

## Question 2: Neural Network Theory (25 points)

**a) Derive the backpropagation algorithm for a single neuron. Show how gradients are calculated and propagated backward through the network.**

**b) Explain the vanishing gradient problem. How do modern architectures like ResNets and LSTMs address this issue?**

**c) What is the difference between batch gradient descent, stochastic gradient descent, and mini-batch gradient descent? When would you use each?**

## Question 3: Optimization and Regularization (25 points)

**a) Explain how the Adam optimizer works. What are its key advantages over basic SGD?**

## Question 4: Model Evaluation (20 points)

**a) For a binary classification problem, explain precision, recall, and F1-score. When would you prefer one metric over another?**

**c) Explain the concept of confusion matrix. How can it be used to calculate accuracy, precision, and recall?**
